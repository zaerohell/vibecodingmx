Cómo trabajo
01
Diagnóstico

Entiendo el negocio, los flujos actuales y los puntos de dolor. Definimos el alcance y la arquitectura antes de escribir una línea de código.

02
Diseño técnico

Schema de datos, API routes, módulos del sistema. Todo documentado en CLAUDE.md + contexto de proyecto antes del primer commit.

03
Desarrollo iterativo

Sprints cortos con entregas reales. Tests automatizados desde el día uno. Deploy continuo para feedback inmediato. Linear + ClickUp para trazabilidad.

04
Producción

CI/CD en verde, monitoring con Sentry, backups automáticos. El sistema vive en la nube y escala con el negocio.
